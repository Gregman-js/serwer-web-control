# serwer-web-control
Serwer web Control based on Symfony PHP, working on ssh layer
You can add many serwers, create command and send it via ajax
Read your machine status, temperature, fan speed, others
Plan a shutdown

## Requirements
* composer
* (L)AMP
    * APACHE2
    * MySql
    * PHP
* bower

## Instalation

`composer install`

`bower install`

`php bin/console d:s:u -f`
